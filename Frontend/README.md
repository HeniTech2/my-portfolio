# 💼 Personal Portfolio Website

A modern, responsive personal portfolio website built using React, TypeScript, and Tailwind CSS.  
It showcases my skills, projects, and contact information in a clean and professional UI.

---

## ✨ Features

- ⚡ Fast and responsive design
- 🎨 Modern UI with Tailwind CSS
- ✨ Smooth animations using Framer Motion
- ⌨️ Typing animation effect
- 📱 Mobile-friendly navigation
- 📩 Contact section integration
- 🌙 Clean modern developer portfolio layout

---

## 🛠 Tech Stack

- React (Vite)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- React Typed
